# Figma Code Viewer Plugin

A simple Figma plugin to show code locations for your components.

## Installation

1. **Open Figma Desktop**
2. Go to **Plugins** → **Development** → **Import plugin from manifest**
3. Select this directory: `/home/memez/quantBot/web/scripts/figma-code-viewer-plugin`
4. Click **Save**

## Usage

1. Select a component in Figma (e.g., SIGN IN component)
2. Run the plugin: **Plugins** → **Development** → **Code Viewer**
3. Click "Get Component Code"
4. See the code file path and live URL

## What It Shows

- Component name
- Code file path (e.g., `/web/components/sign-in.tsx`)
- Live preview URL (e.g., `http://localhost:3000/figma-replicas/sign-in`)

## Note

This is a **manual alternative** to Code Connect (which requires Organization/Enterprise plan).
