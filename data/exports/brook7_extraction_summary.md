## Data Quality

### Case Sensitivity
All Solana mint addresses preserve their original case exactly as they appeared in the messages. Examples:
- `GuhgaLx17yG56Xv6bb6C7Mun9a88RLc7zEEqp5oFpump`

### Timestamp Format

- Example: `2025-12-05T18:53:22.000Z`

## Usage

### Query Database
```sql
-- Get all brook7 calls
SELECT caller_name, token_address, alert_timestamp 
FROM caller_alerts 
ORDER BY alert_timestamp DESC;

-- Get calls by specific caller
SELECT * FROM caller_alerts 
WHERE caller_name = 'exy' 
ORDER BY alert_timestamp DESC;

-- Get recent calls (last 24 hours)
SELECT * FROM caller_alerts 
WHERE alert_timestamp >= datetime('now', '-1 day')
ORDER BY alert_timestamp DESC;
```

### Load CSV in Python
```python
import pandas as pd

# Load all calls
df = pd.read_csv('data/exports/csv/brook7_calls_newest.csv')
```
