# Legacy Scripts Archive

This directory contains archived legacy scripts that have been migrated to the new modular architecture.

## Directory Structure

- `simulation/` - Old simulation scripts (migrated to config-driven engine)
- `optimization/` - Old optimization scripts (migrated to optimization module)
- `analysis/` - Old analysis scripts (migrated to analysis module)
- `reporting/` - Old reporting scripts (migrated to reporting module)
- `data-processing/` - Old data processing scripts (migrated to data modules)

## Migration Status

See `docs/migration/legacy-scripts.md` for the complete migration mapping.

## Note

These scripts are kept for reference but should not be used in production. Use the new modular architecture instead.
